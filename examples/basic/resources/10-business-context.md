# Contexto de Inventiva

Inventiva es un sistema administrativo utilizado para gestionar operaciones internas.

El sistema incluye modulos para ventas, compras, stock, cuentas a cobrar, facturacion, clientes, proveedores y reportes.

## Modulos principales

### Ventas

Permite registrar pedidos, emitir facturas, consultar clientes y revisar cuentas pendientes.

### Compras

Permite registrar compras, proveedores, ordenes de compra y recepcion de mercaderias.

### Stock

Permite consultar existencias, movimientos, ajustes y transferencias entre sucursales.

### Cuentas a Cobrar

Permite consultar saldos de clientes, registrar cobros y revisar cuentas pendientes.

### Caja

Permite registrar cobros, pagos, cierres de caja y arqueos.

### Facturacion

Permite emitir facturas, notas de credito, notas de debito y remisiones.
