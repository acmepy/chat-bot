# Procedimientos operativos

## Consultar stock de un articulo

Ruta:

Stock > Reportes > Existencia de Articulos

Pasos:

1. Ingresar al modulo Stock.
2. Seleccionar Reportes.
3. Abrir Existencia de Articulos.
4. Elegir la sucursal si el sistema lo solicita.
5. Buscar el articulo por codigo, descripcion o codigo de barras.
6. Verificar las cantidades disponible, reservada y comprometida.

Respuesta esperada:

Para consultar el stock de un articulo, ingrese a Stock > Reportes > Existencia de Articulos. Luego busque el articulo por codigo, descripcion o codigo de barras y verifique la cantidad disponible, reservada y comprometida.

No puedo consultar el stock real desde aqui.

## Consultar saldos de clientes

Ruta:

Cuentas a Cobrar > Reportes > Saldos de Clientes a una fecha

Pasos:

1. Ingresar al modulo Cuentas a Cobrar.
2. Seleccionar Reportes.
3. Abrir Saldos de Clientes a una fecha.
4. Indicar la fecha de corte.
5. Buscar o filtrar el cliente si corresponde.
6. Revisar el saldo pendiente.

## Emitir una nota de credito

Ruta:

Ventas > Movimientos > Nota de credito

Pasos:

1. Ingresar al modulo Ventas.
2. Seleccionar Movimientos.
3. Abrir Nota de credito.
4. Buscar la factura relacionada.
5. Verificar cliente, timbrado y condicion.
6. Cargar los items a devolver o corregir.
7. Guardar.
